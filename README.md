## Goal

- No tracker, no proxy
- Multi platform (Linux, Mac OS, Windows)
- Simple and efficient UI, mail debug tools
- Write mail with [Markdown](https://guides.github.com/features/mastering-markdown/) and Hightlighted Code


Install:

    npm install -g mail-up
    mail-up


Start:

    npm start
    npm run dev


Run tests:

    npm test
